Zona Privada

Ha esta zona solo pueden accder usuario logeados

Para porteger el acceso a esta ruta debemos usar un GUARD y definirla en routing

path: 'privado', component: PrivadoComponent, canActivate: [LoginGuard]
canActivate
Guarda para indicar si se puede activar esa ruta
LoginGuard
Guarda creada por nostros, que retornara true o false
